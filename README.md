# squint

A colour difference game. Find the tile that's a slightly different colour — before time runs out.

## How to play

- A grid of colour tiles appears. One tile is a slightly different shade.
- Click the odd one out to score points.
- Faster answers score more points. Higher levels give a bigger multiplier.
- Wrong answer or running out of time costs a life. You have 3 lives.

## Difficulty

The game gets harder as your score increases (every 5 correct answers = new level):

| Level | Grid | Colour delta |
|-------|------|-------------|
| 1 | 3×3 | Large |
| 5 | 4×4 | Medium |
| 10 | 6×6 | Nearly invisible |

The timer also shrinks at higher levels.

## Running locally

Just open `index.html` in a browser — no build step, no dependencies.
